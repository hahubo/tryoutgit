Welcome to dieka.org

DiekA is an open source finite element code (FEM) for forming simulations and evaluating advanced plasticity models for large deformations.

The most important features of the code are:
1. The arbitrary Lagrangian Eulerian method (ALE).
With this method the displacement of the element mesh is uncoupled from the material displacement. Material "flows" through the mesh. ( Huetink), ( Wisselink).
2. The temperature can be used as a degree of freedom.
3. Contact with special interface elements, and by a generalized contact method ( Kloosterman).
4. Tool description can be done by geometric contours, and by external files, Patran neutral files or GID files).
5. For 3 dimensional calculations bulk elements, membrane elements, plate elements and contact elements have been developed.
6. Many material models are available for large deformation plasticity.

You can download a DiekA installation package for Linux, Unix and MS-Windows from : https://dieka.huetink.nl

The package has some simple preprocessors developed by ( Nico van Vliet):

A preprocessor for simulation of Rolling, written in Turbo Pascal (first version 1992 and later on adapted for additional Dieka Features). It can only be executed under a DOS emulator.

A deepdrawing preprocessor written in C.

For more complicates mesh generation you need the commercial MSC-Patran package. The Patran neutral file format can optionally be used for (mesh) input and is the format for postprocessing There is a Patran Student edition . Also the free GID package has limited options for Dieka preprocessing.
The included postprocessing features are usually sufficient for examining the results, Patran and GID have more extensive features.
